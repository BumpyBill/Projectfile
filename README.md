## Projectfile

Run Node Projects Locally With Ease

---

### Usage

```bash
projectfile <dir>
```

### Installation

> Installing Projectfile globally

```bash
npm i projectfile --g
```

> Installing Projectfile locally

```bash
npm i projectfile --save-dev
```

### Projectfile

##### Environment Variables

```dockerfile
ENV <name> <value>
```

##### Environment Variables

```dockerfile
CMD <string>
```
